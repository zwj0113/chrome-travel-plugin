🔄 核心数据流

视频类页面：
用户点击 → 适配器提取视频URL/元数据 → 下载音频 → 硅基流动转录(失败→Whisper) → AI总结 → 抓取评论 → 生成MD → 侧边栏预览+下载

图文类页面：
用户点击 → 适配器提取图片/文本/元数据 → DeepSeek图片理解+内容总结 → 抓取评论 → 生成MD → 侧边栏预览+下载
